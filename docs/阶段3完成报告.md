## 6. FastAPI 接口说明

### 6.1 `GET /api/media/health`（新增）

**入参：** 无

**出参示例：**

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "status": "ok",
    "data_platform_enabled": true,
    "data_platform_base_url": "http://127.0.0.1:8000",
    "data_platform_ok": true,
    "data_platform": {"status": "ok", "service": "ai-query"},
    "default_media_room_id": "ChatRoom01",
    "media_store": {
      "ttl_sec": 120,
      "max_per_room": 5,
      "rooms": {}
    }
  }
}
```

| 字段 | 说明 |
|------|------|
| data_platform_enabled | 是否开启中台优先 |
| data_platform_ok | 启用时中台 health 是否成功；未启用则为 null |
| media_store | 当前各房间队列长度等 |

**数据流：**

```
前端/运维 GET /api/media/health
  → 若 ENABLED：GET {中台}/api/v1/ai/health
  → 汇总 JSON 返回
```
